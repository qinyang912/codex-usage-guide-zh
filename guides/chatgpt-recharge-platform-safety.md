## 七项平台选择标准

### 1. 账号归属

优先让套餐进入自己长期控制的 ChatGPT 账号。共享号、成品号、临时工作区和自己的账号直充不是同一种商品，不能直接比较价格。

### 2. 商品类型

确认购买的是 Go、Plus、Pro、Business 席位、ChatGPT Credits 还是 Platform API 余额。ChatGPT 订阅与 OpenAI API 属于不同计费系统；Plus 费用不会自动变成 API 余额。

### 3. 真实收款方

OpenAI、Apple、Google、独立网站和电商商家负责不同的账单与退款流程。宣传页写什么不如订单和收据显示的收款方重要。

### 4. 凭证范围

密码、邮箱验证码、MFA 验证码、恢复码、Cookie、Session 和 API Key 都是敏感凭证。服务方应说明为什么需要、用多久、何时删除，以及是否有更少权限的方案。

### 5. 订阅管理

充值后应知道去哪里看续费、取消、到期日期和 Invoice。若服务方说不清网页、Apple、Google 或第三方订单中的哪条路径负责后续管理，售后边界就不清楚。

### 6. 失败与退款

下单前写明处理时间、失败判定、退款方式和预计入账时间。“不到账退款”还需要明确谁判定不到账，以及退款退回原支付方式还是平台余额。

### 7. 可验证证据

持续维护的帮助内容、明确的主体与政策、真实订单记录和长期售后入口，比“服务十万用户”“永不掉订阅”“百分之百不封号”等无法核验的口号更有价值。

## 代充为什么可能比某些页面便宜

价格偏低不必然等于有问题，价格接近官方标价也不证明一定安全。先在同套餐、同周期、同账号归属下比较。

- 权益出现后又通过争议退款、撤销付款等方式回收成本。

普通用户很难仅凭到账速度判断付款来源。更稳妥的做法是保留订单、确认订阅管理渠道、避免共享账号和高权限凭证，并要求对异常价差作书面解释，而不是自行把所有低价交易都判定为“黑充”。
