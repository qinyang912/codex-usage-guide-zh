# ChatGPT Plus 免费试用、首月免费和促销码核对指南

> 核对日期：2026-07-21。活动资格、地区和续费规则可能变化，结账页与账号内显示优先。

Google 上的“ChatGPT Plus 免费试用”“首月免费”和“学生优惠”经常指向不同活动。OpenAI 会通过合作伙伴、邮件或产品内入口发放特定促销，但这不等于每个账号都存在一个长期、通用的 Plus 免费月。

## 先判断你看到的是哪一种优惠

| 页面写法 | 可能对应 | 核对重点 |
| --- | --- | --- |
| 免费试用 | 特定活动的限时试用 | 资格、试用周期、是否要求付款方式 |
| 首月免费 | 结账时首个周期价格为 0 | 下一次续费日期和标准月费 |
| 折扣码 / promo code | 合作伙伴或推荐活动 | 发行方、有效期、地区与新老用户限制 |
| 学生优惠 | 地区和时间受限的学生项目 | 当前活动页面、学校验证与权益类型 |
| “免费使用 ChatGPT” | ChatGPT Free 套餐 | 不应自动理解为 Plus 免费试用 |

## 兑换前检查 8 项

1. 入口是否来自 OpenAI、可信合作伙伴邮件或产品内提示。
2. 当前账号是否属于活动面向的新用户、免费用户或指定人群。
3. 活动是否支持账号所在地区和付款方式。
4. 代码或链接是否仍在有效期内。
5. 结账页是否真的显示 `0.00` 或明确的折后金额。
6. 优惠周期结束后会不会自动续订，标准月费是多少。
7. 取消后权益是立即结束，还是持续到当前周期结束。
8. 是否有人索要密码、验证码、Session、Cookie 或 API Key。

只看广告标题不够。OpenAI 官方说明提到，促销可能是免费试用、折扣或一次性优惠价；适用对象由具体活动决定。代码通常只能使用一次，不能叠加，也不应被复制、出售或转让。

## 为什么会提示“不符合资格”

常见原因包括：账号已经订阅 Plus、过去使用过相似活动、代码过期、地区不匹配，或者付款方式不满足活动要求。合作伙伴发放的代码如果失效，应先联系发放方；OpenAI 通常不会重新补发合作伙伴代码。

## 自动续费和取消

大多数促销在优惠期结束后会按标准月费自动续订，除非在续费前取消。官方建议至少提前 24 小时处理取消。网页订阅通常从 `Settings → Subscription → Manage → Cancel` 操作；Apple 或 Google 渠道订阅则应在对应商店取消。

免费活动在取消时可能立即结束，折扣订阅则可能保留到已经支付的周期结束，具体仍以活动条款和取消页面提示为准。使用过的优惠码通常不能在重新订阅时再次恢复。

## 学生优惠要看“当前活动”，不要套用旧截图

OpenAI 当前的 Codex 学生页面面向符合条件的美国、加拿大高校学生，提供的是用于 Codex 超额使用的 ChatGPT Credits，并不是 API Credits，也不能据此推断全球学生都能免费获得 Plus。网上过去的 Plus 学生活动截图可能已经过期，核对时要看活动页面的地区、验证方式和有效期。

## 防骗提醒

- 不购买来源不明、承诺“永久有效”或“所有账号通用”的代码。
- 不在陌生网站输入 ChatGPT 密码、邮箱验证码或长期登录凭证。
- 不因为结账页要求绑定付款方式，就忽略下一次续费金额和日期。
- 不把商家聊天优惠码当作可以在 ChatGPT 官方结账页兑换的 OpenAI 促销码。
