# 2026 ChatGPT 学生优惠：Plus、Codex Credits 与 Edu 怎么区分

> 核对日期：2026-07-21。活动资格、金额与有效期会变化，申请前请重新打开 OpenAI 官方页面。

## 先看结论

目前能从 OpenAI 官方直接核验的大学生权益，是向符合资格的美国、加拿大在校大学生提供 **100 美元 Codex Credits**。官方 Usage 页面应显示 **2,500 Credits**。

它不是：

- 面向所有学生的免费 ChatGPT Plus；
- OpenAI Platform API Credits；
- 可以转卖或共享的通用优惠码；
- 自动向所有教育邮箱开放的 ChatGPT Edu 席位。

## 当前学生 Codex Credits 资格

OpenAI 当前页面列出的条件包括：

1. 当前就读于美国或加拿大的授位大学；
2. 申领时居住在美国或加拿大；
3. 通过 SheerID 验证学生身份和活动资格；
4. 每名学生只能领取一次；
5. 需要一个 ChatGPT Free、Go、Plus 或 Pro 账号；
6. 可以使用个人 ChatGPT 账号，但验证仍需要大学邮箱。

中国大学邮箱、教育邮箱后缀或学生证本身，不会自动满足美国、加拿大的学校与居住地要求。

## 100 美元 / 2,500 Credits 怎么用

这些 Credits 用于原 ChatGPT 套餐包含用量耗尽后的 Codex 超额使用：

| 可以 | 不可以 |
| --- | --- |
| 抵扣受支持的 Codex 额外用量 | 把 Free 自动升级为 Plus |
| 发放到个人 ChatGPT 工作区 | 支付 OpenAI Platform API 账单 |
| 在 Usage 页面查看余额 | 当作永久或可转卖余额 |

官方当前说明，额度自发放日起 **12 个月后过期**。账号有多个工作区时，额度会添加到个人工作区。

## 申请和验收顺序

1. 从 OpenAI 官方大学生 Codex 页面进入活动。
2. 登录准备接收额度的个人 ChatGPT 账号。
3. 通过 SheerID 完成学生身份验证。
4. 验证通过后等待额度自动发放。
5. 打开 Usage 页面，确认显示 2,500 Credits。
6. 记录发放日期和 12 个月后的到期时间。

不要通过卖家或陌生表单提交学生证、学校邮箱密码、ChatGPT 密码、验证码、Session 或 Cookie。

## Plus 学生优惠、Codex Credits 与 ChatGPT Edu

| 名称 | 实际含义 | 如何确认 |
| --- | --- | --- |
| ChatGPT Free | 长期免费个人套餐 | 官方 Pricing 与账号套餐页 |
| ChatGPT Plus | 个人月度付费套餐 | 账号结算页显示的实际价格 |
| 学生 Codex Credits | 符合资格学生的 Codex 超额用量 | 官方学生页、SheerID、Usage |
| ChatGPT Edu | 由学校采购和管理的教育工作区 | 学校 IT、SSO 与正式通知 |
| 促销 / Referral | 某次活动中的限时折扣或试用 | 原始活动条款和账号结算页 |

学校邮箱能登录 ChatGPT，不等于学校已经部署 ChatGPT Edu。历史学生 Plus 活动、旧截图或“2026 最新”教程，也不能证明当前账号仍符合资格。
